Question 1:For the three categories of web software technology we've just defined, you are asked to find out which category so-called "Chrome plug-in" technology belongs to. Please describe the basic idea of the technology, and give your reasoning on its position in the technology categories.

答：Chrome插件属于productivity and quality-oriented technologies一类。利用Java、Javascript等技术可使浏览器不仅仅局限于简单地呈现服务器所返回的内容，还能实现用户与页面以及服务后端的交互。Chrome插件实际上就是一个Web应用，然后将按照Chrome的规定将一个快捷方式放在Chrome工具栏上。

Question 2:In a normal intercommunication between Web browser and Web Server, Web Server closes the network communication right after it sends back the document requested by the browser. Think about this statusless character of World Wide Web. Give some good points and bad points of it. Can a web server know exactly how many browsers are reading its document?

答:无状态服务是对单次请求的处理，不依赖其他请求，处理一次请求所需的全部信息，都包含在请求里，服务器本身不存储任何信息。其优点在于服务器的工作量较小且不需要存储额外的信息，缺点是无法向有状态服务那样实现与服务器的实时通信。
